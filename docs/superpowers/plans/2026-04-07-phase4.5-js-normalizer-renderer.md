# Phase 4.5: JS Validator + Normalizer + Renderer — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the Python normalizer and HTML renderer to TypeScript in `clearnotation-js/` so the browser editor can produce publication-quality HTML export without a server round-trip.

**Architecture:** This is a behavioral port — the Python code defines exact behavior, and the TypeScript version must produce identical HTML output for every valid fixture. The port covers three modules: (1) normalized AST types (NHeading, NParagraph, NCallout, etc.), (2) normalizer (walks the converter's BNBlock tree, assigns IDs via slugification, numbers notes, resolves table cells, produces a NormalizedDocument), and (3) HTML renderer (walks the normalized AST and produces semantic HTML5). The normalizer consumes `BNBlock[]` from the converter module (Phase 3), not the Python parser's `Document` model. This means the normalizer's input types differ from Python, but the output (NormalizedDocument) and HTML are identical.

**Tech Stack:** TypeScript 5.3+, Vitest 4+, existing `editor/src/converter/` (BNBlock, BNInlineContent types), existing `editor/src/schema/` (registry)

## Scope

**Porting from Python:**
- `models.py` (lines 154-296): Normalized AST types — NHeading, NParagraph, NThematicBreak, NBlockQuote, NUnorderedList, NOrderedList, NToc, NCallout, NFigure, NMathBlock, NTable, NSourceBlock, NRef, NormalizedDocument

- `renderer.py` (220 lines): render_html function — full HTML5 document, block rendering, inline rendering, TOC, footnotes, tables
- `utils.py:split_table_row` (33 lines): Table cell splitting with escape handling

**NOT porting:**
- `validator.py`: File-system concerns (include resolution, config discovery). The browser editor validates at the schema level (already done in Phase 2).
- `parser.py`, `inline_parser.py`: The browser uses tree-sitter + the converter from Phase 3 instead.

**Key difference from Python:** The normalizer's input is `BNBlock[]` (from the converter), not `Document` (from the Python parser). The normalizer must walk BNBlock's flat inline content model (`BNStyledText`/`BNLink` with style flags) instead of Python's nested inline nodes (`Strong(children=[...])`). The output NormalizedDocument and HTML are identical.

```typescript
/**
 * Generate a URL-friendly slug from text.
 * Matches the Python normalizer's slugify behavior:
 * NFKD normalize → lowercase → keep ASCII alnum → collapse non-alnum to dash → strip edges.
 */
export function slugify(text: string): string {
  const normalized = text.normalize("NFKD").toLowerCase();
  const chars: string[] = [];
  let lastDash = false;
  for (const ch of normalized) {
    const code = ch.charCodeAt(0);
    if (code < 128 && isAlnum(ch)) {
      chars.push(ch);
      lastDash = false;
    } else if (!lastDash) {
      chars.push("-");
      lastDash = true;
    }
  }
  return chars.join("").replace(/^-+|-+$/g, "");
}

function isAlnum(ch: string): boolean {
  return (ch >= "a" && ch <= "z") || (ch >= "0" && ch <= "9");
}

/**
 * Split a pipe-delimited table row into cell strings.
 * Handles \| (escaped pipe) and \\ (escaped backslash).
 * Unknown escape sequences are preserved (backslash kept).
 * Matches Python utils.split_table_row behavior.
 */
export function splitTableRow(line: string): string[] {
  const cells: string[] = [];
  const current: string[] = [];
  let i = 0;
  while (i < line.length) {
    const ch = line[i];
    if (ch === "\\") {
      i++;
      if (i < line.length && (line[i] === "|" || line[i] === "\\")) {
        current.push(line[i]);
        i++;
        continue;
      }
      current.push("\\");
      continue;
    }
    if (ch === "|") {
      cells.push(current.join("").trim());
      current.length = 0;
      i++;
      continue;
    }
    current.push(ch);
    i++;
  }
  cells.push(current.join("").trim());
  return cells;
}

/**
 * Escape HTML special characters.
 * Matches Python html.escape(text, quote=True).
 */
export function escHtml(text: string): string {
  return text
    .replace(/&/g, "&amp;")
    .replace(/</g, "&lt;")
    .replace(/>/g, "&gt;")
    .replace(/"/g, "&quot;")
    .replace(/'/g, "&#x27;");
}
```

**Files:**
- Create: `clearnotation-js/src/normalizer.ts`

**Context:** The normalizer takes `BNBlock[]` (from Phase 3 converter) and produces `NormalizedDocument`. The hardest part is converting BlockNote's flat styled-text model (`BNStyledText` with `styles: { clnStrong: true }`) back to the Python-style nested inline model (`NStrong { children: [NText] }`). This is the reverse of what the converter does — a "flat-to-tree" reconstruction.

**BNInlineContent → NormalizedInline mapping:**
- `BNStyledText` with no marks → `NText { value }`
- `BNStyledText` with `clnCode: true` → `NCodeSpan { value }`
- `BNStyledText` with `clnRef: "target"` → `NRef { target }`
- `BNStyledText` with `clnStrong: true` → must be wrapped in `NStrong`
- `BNStyledText` with `clnEmphasis: true` → must be wrapped in `NEmphasis`
- `BNStyledText` with `clnNote: true` → must be wrapped in `NNote` (with number)
- `BNLink` → `NLink { label, target }`

The flat-to-tree algorithm is the same as the serializer's style-to-tree, but producing AST nodes instead of text. Priority: clnNote > clnStrong > clnEmphasis > clnCode > clnRef.

```typescript
import type { BNBlock, BNInlineContent, BNStyledText, BNLink } from "../../editor/src/converter/types";
import type {
  NormalizedInline, NormalizedBlock, NormalizedDocument,
  NText, NCodeSpan, NStrong, NEmphasis, NLink, NNote, NRef,
  NHeading, NParagraph, NThematicBreak, NBlockQuote,
  NUnorderedList, NOrderedList, NOrderedItem, NToc,
  NCallout, NFigure, NMathBlock, NTable, NTableRow, NTableCell,
  NSourceBlock,
} from "./types";
import { slugify, splitTableRow } from "./utils";

/** Mark priority for flat-to-tree reconstruction (outermost first). */
const MARK_PRIORITY = ["clnNote", "clnStrong", "clnEmphasis"] as const;

/**
 * Normalize an array of BNBlocks into a NormalizedDocument.
 * This is the JS equivalent of Python's Normalizer.normalize().
 */
export function normalize(
  blocks: BNBlock[],
  meta?: Record<string, unknown>
): NormalizedDocument {
  const state: NormalizerState = {
    noteCounter: 1,
    notes: [],
    slugCounts: {},
    meta: meta ?? {},
  };

  const normalized = normalizeBlocks(blocks, state);

  return {
    meta: state.meta,
    blocks: normalized,
    notes: state.notes,
  };
}

interface NormalizerState {
  noteCounter: number;
  notes: NNote[];
  slugCounts: Record<string, number>;
  meta: Record<string, unknown>;
}

function normalizeBlocks(blocks: BNBlock[], state: NormalizerState): NormalizedBlock[] {
  const result: NormalizedBlock[] = [];
  let pendingAnchor: string | null = null;

  for (const block of blocks) {
    // Meta block: merge into document meta, don't emit as a block
    if (block.type === "clnMeta") {
      const entries = JSON.parse((block.props.entries as string) || "{}");
      Object.assign(state.meta, entries);
      continue;
    }

    // Anchor: consume and apply to next block
    if (block.type === "clnAnchor") {
      pendingAnchor = block.props.id as string;
      continue;
    }

    // Include: skip (unsupported in browser)
    if (block.type === "clnInclude") {
      continue;
    }

    const normalized = normalizeBlock(block, state, pendingAnchor);
    if (normalized) {
      result.push(normalized);
    }
    pendingAnchor = null;
  }

  return result;
}

function normalizeBlock(
  block: BNBlock,
  state: NormalizerState,
  pendingAnchor: string | null
): NormalizedBlock | null {
  switch (block.type) {
    case "clnHeading": {
      const content = normalizeInlines(block.content, state);
      const id = pendingAnchor ?? generateSlug(block.content, state);
      return { type: "heading", level: block.props.level as number, id, content };
    }

    case "clnParagraph": {
      const content = normalizeInlines(block.content, state);
      const result: NParagraph = { type: "paragraph", content };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnThematicBreak":
      return { type: "thematic_break" };

    case "clnCodeBlock": {
      const result: NSourceBlock = {
        type: "source_block",
        language: (block.props.language as string) || "",
        text: (block.props.code as string) || "",
      };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnBlockquote": {
      const lines = splitContentByNewlines(block.content, state);
      const result: NBlockQuote = { type: "blockquote", lines };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnUnorderedList": {
      // Each clnUnorderedList block is ONE item; collect consecutive ones
      // But the normalizer receives them individually, so wrap as single-item list
      const items = [normalizeInlines(block.content, state)];
      const result: NUnorderedList = { type: "unordered_list", items };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnOrderedList": {
      const ordinal = (block.props.startNumber as number) || 1;
      const content = normalizeInlines(block.content, state);
      const items: NOrderedItem[] = [{ ordinal, content }];
      const result: NOrderedList = { type: "ordered_list", items };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnToc": {
      const result: NToc = { type: "toc" };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnCallout": {
      const childBlocks = normalizeBlocks(block.children, state);
      const result: NCallout = {
        type: "callout",
        kind: (block.props.kind as string) || "",
        title: (block.props.title as string) || null,
        compact: (block.props.compact as boolean) || false,
        blocks: childBlocks,
      };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnFigure": {
      const childBlocks = normalizeBlocks(block.children, state);
      const result: NFigure = {
        type: "figure",
        src: (block.props.src as string) || "",
        blocks: childBlocks,
      };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnMath": {
      const result: NMathBlock = {
        type: "math_block",
        text: (block.props.rawContent as string) || "",
      };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnTable": {
      const result = normalizeTable(block, state);
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    case "clnSource": {
      const result: NSourceBlock = {
        type: "source_block",
        language: (block.props.language as string) || "",
        text: (block.props.rawContent as string) || "",
      };
      if (pendingAnchor) result.id = pendingAnchor;
      return result;
    }

    default:
      return null;
  }
}

function normalizeTable(block: BNBlock, state: NormalizerState): NTable {
  const rawData: string[][] = JSON.parse((block.props.tableData as string) || "[]");
  const rows: NTableRow[] = rawData.map((rawRow) => ({
    cells: rawRow.map((cellText) => ({
      content: [{ type: "text" as const, value: cellText }],
    })),
  }));

  return {
    type: "table",
    header: (block.props.header as boolean) || false,
    align: null, // align is encoded in tableData for the editor
    rows,
  };
}

/**
 * Split BNInlineContent[] by newline text nodes into lines.
 * Used for blockquotes which store multiple lines in one content array.
 */
function splitContentByNewlines(
  content: BNInlineContent[],
  state: NormalizerState
): NormalizedInline[][] {
  const lines: NormalizedInline[][] = [[]];
  for (const item of content) {
    if (item.type === "text" && item.text === "\n") {
      lines.push([]);
    } else {
      const normalized = normalizeInlineItem(item, state);
      lines[lines.length - 1].push(...normalized);
    }
  }
  return lines;
}

/**
 * Convert flat BNInlineContent[] to nested NormalizedInline[].
 * This is the core flat-to-tree reconstruction.
 */
function normalizeInlines(
  content: BNInlineContent[],
  state: NormalizerState
): NormalizedInline[] {
  const result: NormalizedInline[] = [];
  let i = 0;

  while (i < content.length) {
    const item = content[i];

    // Links
    if (item.type === "link") {
      const label = normalizeInlines(item.content, state);
      result.push({ type: "link", label, target: item.href });
      i++;
      continue;
    }

    // Ref (string style value)
    if (typeof item.styles.clnRef === "string") {
      result.push({ type: "ref", target: item.styles.clnRef });
      i++;
      continue;
    }

    // Code (no nesting inside code)
    if (item.styles.clnCode === true) {
      result.push({ type: "code_span", value: item.text });
      i++;
      continue;
    }

    // Find the outermost mark
    const outerMark = findOuterMark(item.styles);

    if (!outerMark) {
      // Plain text — handle newlines
      if (item.text === "\n") {
        // Newline nodes are kept as-is for now
        result.push({ type: "text", value: "\n" });
      } else {
        result.push({ type: "text", value: item.text });
      }
      i++;
      continue;
    }

    // Group consecutive items sharing this mark
    const group: BNInlineContent[] = [];
    while (i < content.length && itemHasMark(content[i], outerMark)) {
      group.push(content[i]);
      i++;
    }

    // Strip the mark from grouped items and recurse
    const stripped = group.map((g) => stripMark(g, outerMark));
    const children = normalizeInlines(stripped, state);

    if (outerMark === "clnNote") {
      const note: NNote = {
        type: "note",
        children,
        number: state.noteCounter++,
      };
      state.notes.push(note);
      result.push(note);
    } else if (outerMark === "clnStrong") {
      result.push({ type: "strong", children });
    } else if (outerMark === "clnEmphasis") {
      result.push({ type: "emphasis", children });
    }
  }

  return result;
}

function normalizeInlineItem(
  item: BNInlineContent,
  state: NormalizerState
): NormalizedInline[] {
  return normalizeInlines([item], state);
}

function findOuterMark(styles: Record<string, boolean | string>): string | null {
  for (const mark of MARK_PRIORITY) {
    if (styles[mark] === true) return mark;
  }
  return null;
}

function itemHasMark(item: BNInlineContent, mark: string): boolean {
  if (item.type === "link") {
    return item.content.every((t) => t.styles[mark] === true);
  }
  if (typeof (item as BNStyledText).styles.clnRef === "string") return false;
  return (item as BNStyledText).styles[mark] === true;
}

function stripMark(item: BNInlineContent, mark: string): BNInlineContent {
  if (item.type === "link") {
    return {
      ...item,
      content: item.content.map((t) => ({
        ...t,
        styles: withoutKey(t.styles, mark),
      })),
    };
  }
  return { ...item, styles: withoutKey((item as BNStyledText).styles, mark) };
}

function withoutKey(
  obj: Record<string, boolean | string>,
  key: string
): Record<string, boolean | string> {
  const copy = { ...obj };
  delete copy[key];
  return copy;
}

/**
 * Generate a slug for a heading from its inline content.
 * Extracts plain text, slugifies, deduplicates.
 */
function generateSlug(content: BNInlineContent[], state: NormalizerState): string {
  const text = plainText(content);
  const base = slugify(text);
  const count = (state.slugCounts[base] || 0) + 1;
  state.slugCounts[base] = count;
  return count === 1 ? base : `${base}-${count}`;
}

function plainText(content: BNInlineContent[]): string {
  return content
    .map((item) => {
      if (item.type === "link") return plainText(item.content);
      if (typeof item.styles.clnRef === "string") return "";
      if (item.styles.clnNote === true) return "";
      return item.text;
    })
    .join("");
}
```

```typescript
import type {
  NormalizedDocument, NormalizedBlock, NormalizedInline,
  NHeading, NParagraph, NThematicBreak, NBlockQuote,
  NUnorderedList, NOrderedList, NToc, NCallout, NFigure,
  NMathBlock, NTable, NSourceBlock, NNote,
} from "./types";
import { escHtml } from "./utils";

export interface RenderOptions {
  cssPath?: string;
}

/**
 * Render a NormalizedDocument to a complete HTML5 string.
 * Matches Python renderer.render_html() output.
 */
export function renderHtml(
  doc: NormalizedDocument,
  options?: RenderOptions
): string {
  const cssPath = options?.cssPath ?? "clearnotation.css";
  const title = typeof doc.meta.title === "string" ? doc.meta.title : "ClearNotation Document";

  const headings = doc.blocks.filter((b): b is NHeading => b.type === "heading");

  const parts: string[] = [];
  parts.push("<!DOCTYPE html>");
  parts.push('<html lang="en">');
  parts.push("<head>");
  parts.push('<meta charset="utf-8">');
  parts.push(`<title>${escHtml(title)}</title>`);
  parts.push(`<link rel="stylesheet" href="${escHtml(cssPath)}">`);
  parts.push("</head>");
  parts.push("<body>");

  for (const block of doc.blocks) {
    parts.push(renderBlock(block, headings));
  }

  if (doc.notes.length > 0) {
    parts.push('<hr class="footnotes-sep">');
    parts.push('<section class="footnotes">');
    parts.push("<ol>");
    for (const note of doc.notes) {
      parts.push(`<li id="fn-${note.number}">`);
      parts.push(renderInlines(note.children));
      parts.push(` <a href="#fnref-${note.number}" class="footnote-backref">\u21a9</a>`);
      parts.push("</li>");
    }
    parts.push("</ol>");
    parts.push("</section>");
  }

  parts.push("</body>");
  parts.push("</html>");
  return parts.join("\n");
}

function renderBlock(block: NormalizedBlock, headings: NHeading[]): string {
  switch (block.type) {
    case "heading": {
      const tag = `h${block.level}`;
      return `<${tag} id="${escHtml(block.id)}">${renderInlines(block.content)}</${tag}>`;
    }

    case "paragraph": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      return `<p${attrs}>${renderInlines(block.content)}</p>`;
    }

    case "thematic_break":
      return "<hr>";

    case "blockquote": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      const lines = block.lines.map((line) => `<p>${renderInlines(line)}</p>`).join("\n");
      return `<blockquote${attrs}>\n${lines}\n</blockquote>`;
    }

    case "unordered_list": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      const items = block.items.map((item) => `<li>${renderInlines(item)}</li>`).join("\n");
      return `<ul${attrs}>\n${items}\n</ul>`;
    }

    case "ordered_list": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      const start = block.items[0]?.ordinal ?? 1;
      const startAttr = start !== 1 ? ` start="${start}"` : "";
      const items = block.items
        .map((item) => `<li>${renderInlines(item.content)}</li>`)
        .join("\n");
      return `<ol${attrs}${startAttr}>\n${items}\n</ol>`;
    }

    case "toc": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      return `<nav class="toc"${attrs}>\n${renderToc(headings)}\n</nav>`;
    }

    case "callout": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      const parts: string[] = [`<aside class="callout callout-${escHtml(block.kind)}"${attrs}>`];
      if (block.title) {
        parts.push(`<p class="callout-title">${escHtml(block.title)}</p>`);
      }
      for (const child of block.blocks) {
        parts.push(renderBlock(child, headings));
      }
      parts.push("</aside>");
      return parts.join("\n");
    }

    case "figure": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      const parts: string[] = [`<figure${attrs}>`];
      parts.push(`<img src="${escHtml(block.src)}" alt="">`);
      if (block.blocks.length > 0) {
        parts.push("<figcaption>");
        for (const child of block.blocks) {
          parts.push(renderBlock(child, headings));
        }
        parts.push("</figcaption>");
      }
      parts.push("</figure>");
      return parts.join("\n");
    }

    case "math_block": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      return `<pre class="math"${attrs}><code>${escHtml(block.text)}</code></pre>`;
    }

    case "table":
      return renderTable(block);

    case "source_block": {
      const attrs = block.id ? ` id="${escHtml(block.id)}"` : "";
      return `<pre${attrs}><code class="language-${escHtml(block.language)}">${escHtml(block.text)}</code></pre>`;
    }

    default:
      return "";
  }
}

function renderInlines(inlines: NormalizedInline[]): string {
  return inlines
    .map((node) => {
      switch (node.type) {
        case "text":
          return escHtml(node.value);
        case "code_span":
          return `<code>${escHtml(node.value)}</code>`;
        case "strong":
          return `<strong>${renderInlines(node.children)}</strong>`;
        case "emphasis":
          return `<em>${renderInlines(node.children)}</em>`;
        case "link":
          return `<a href="${escHtml(node.target)}">${renderInlines(node.label)}</a>`;
        case "note":
          return `<sup><a href="#fn-${node.number}" id="fnref-${node.number}" class="footnote-ref">[${node.number}]</a></sup>`;
        case "ref":
          return `<a href="#${escHtml(node.target)}">${escHtml(node.target)}</a>`;
        default:
          return "";
      }
    })
    .join("");
}

function renderToc(headings: NHeading[]): string {
  if (headings.length === 0) return "";
  const items = headings.map((h) => {
    const indent = "  ".repeat(h.level - 1);
    const label = renderInlines(h.content);
    return `${indent}<li><a href="#${escHtml(h.id)}">${label}</a></li>`;
  });
  return "<ul>\n" + items.join("\n") + "\n</ul>";
}

function renderTable(table: NTable): string {
  const attrs = table.id ? ` id="${escHtml(table.id)}"` : "";
  const parts: string[] = [`<table${attrs}>`];

  const rows = [...table.rows];
  if (table.header && rows.length > 0) {
    const headerRow = rows.shift()!;
    parts.push("<thead>");
    parts.push("<tr>");
    for (let i = 0; i < headerRow.cells.length; i++) {
      const style = alignStyle(table.align, i);
      parts.push(`<th${style}>${renderInlines(headerRow.cells[i].content)}</th>`);
    }
    parts.push("</tr>");
    parts.push("</thead>");
  }

  if (rows.length > 0) {
    parts.push("<tbody>");
    for (const row of rows) {
      parts.push("<tr>");
      for (let i = 0; i < row.cells.length; i++) {
        const style = alignStyle(table.align, i);
        parts.push(`<td${style}>${renderInlines(row.cells[i].content)}</td>`);
      }
      parts.push("</tr>");
    }
    parts.push("</tbody>");
  }

  parts.push("</table>");
  return parts.join("\n");
}

function alignStyle(align: string[] | null, index: number): string {
  if (align && index < align.length) {
    return ` style="text-align: ${align[index]}"`;
  }
  return "";
}
```
